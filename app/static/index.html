    query auditavel

    Descreva uma pergunta, gere uma query revisavel, recupere estrutura e sintetize apenas o grafo retornado.

    1. Descrever Linguagem natural vira uma intencao estrutural clara.
    2. Consultar Cypher read-only retorna tabela e, com RETURN p, subgrafo.
    3. Inspecionar Caminhos, vizinhos e pesos viram o material de message passing.
    4. Explicar A LLM usa somente nos, arestas, caminhos e linhas retornadas.
    O LLM gera a query; voce revisa antes de rodar.
